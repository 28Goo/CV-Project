# CV Creator

A React App that generates a Curriculum Vitae by just filling out a form.

## Getting Started

### Prerequisite

-   [Node.js](https://nodejs.org/en/)

```
git clone https://github.com/28Goo/CV-Project.git
cd CV-Project
npm install
npm start
```
